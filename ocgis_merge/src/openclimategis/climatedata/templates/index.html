Welcome to OpenClimateGIS, a geospatial web service that provides access to climate model forecasts in geospatial formats. The goal of OpenClimateGIS is to make climate model datasets readily available in commonly used, modern geospatial formats used by GIS software, browser-based mapping tools, and virtual globes.
OpenClimateGIS implements a REST-like API for requesting data resources, with relevant parameters encoded into the URL path. The OpenClimateGIS system does not store climate data archives locally, but rather works in conjunction with external climate archives that expose climate data via the OPeNDAP protocol.
